Was wir im Labor herausfinden möchten:
--------------------------------------

- Ist der Aufbau vom UI verständlich:
  * Begreifen die Anwender die Konzepte?
  * Können sie sich im UI orientieren?
- Können die Anwender effizient arbeiten?
- Wie schnell können Anwender konkrete Aufgaben lösen?
- Wie fällt der Vergleich zum alten Rayo UI aus?
- Wie kommen die Anwender ans Ziel?
  * Benutzen sie die Maus, Tastatur?
  * Welche Buttons klicken sie häufig, welche nicht?
- Wie ist der Gesamteindruck? Fühlt der Anwender sich gut bei der Arbeit?



Schwierigkeit:
--------------------------------------

  abgeleitet werden können.
- Ergebnisse sollen mit vernünftigem Aufwand auszuwerten sein.
